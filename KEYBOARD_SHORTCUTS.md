# Горячие клавиши (Keyboard Shortcuts)

Система поддерживает быструю навигацию с помощью клавиатурных сокращений.

## Основная команда

### `Cmd+K` / `Ctrl+K`
Открывает командное меню (Command Palette) для быстрого поиска и навигации.

- **Mac**: `⌘ + K`
- **Windows/Linux**: `Ctrl + K`

В командном меню можно:
- Набирать текст для поиска разделов
- Использовать стрелки ↑↓ для навигации
- Нажать `Enter` для выбора

## Прямые шорткаты

Эти команды работают из любого места приложения (кроме полей ввода):

### `Cmd+H` / `Ctrl+H`
Переход на главную страницу

### `Cmd+A` / `Ctrl+A`
Переход в раздел "Анализ угроз"

### `Cmd+G` / `Ctrl+G`
Переход к "Графикам и аналитике"

### `Cmd+?` / `Ctrl+?`
Открыть справку (на главной странице)

## Как использовать

1. **Откройте командное меню**: Нажмите `Cmd+K` (Mac) или `Ctrl+K` (Windows/Linux)
2. **Используйте прямые шорткаты**: Просто нажмите комбинацию клавиш (например, `Cmd+A` для анализа)
3. **Навигация в меню**: 
   - Используйте стрелки для перемещения
   - Введите текст для фильтрации
   - Нажмите Enter для выбора

## Примечания

- Шорткаты **не работают**, когда фокус находится в полях ввода (input, textarea)
- Это сделано для предотвращения конфликтов с обычным вводом текста
- На Mac используется `Cmd`, на Windows/Linux - `Ctrl`
